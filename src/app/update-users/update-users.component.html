Refresh member list

The list of members (used for searching) can be refreshed here. Note that member refreshes consume a significant quantity of resources and should not be performed more than once per day. To perform a member refresh:

  1. Visit the MyTurn site as an administrator
  2. Click Members, then Search/Export
  3. Under Fields to Export, uncheck Select All, then select: First Name, Last Name, Email
  4. By Export results, select CSV
  5. Once the file has downloaded, import it here:
